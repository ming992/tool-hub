桌面智能体安全边界

国内小白使用 WorkBuddy 这类桌面智能体时，最重要的不是“能不能自动操作”，而是“哪些不能让它碰”。

不要授权隐私目录

不要让智能体随意访问微信文件、身份证照片、网银截图、合同原件、公司机密文件夹。

不要让它操作付款

涉及购买、充值、扫码、付款、登录企业系统时，必须人工确认，不让 AI 自动点击。

不要复制陌生命令

遇到删除、格式化、清空、权限提升命令，先暂停。小白只执行本产品给出的低风险检查命令。

复制给 WorkBuddy / CodeBuddy 的任务模板

用户不会表达需求，就让他复制模板。模板必须具体、边界清楚、能验收。
