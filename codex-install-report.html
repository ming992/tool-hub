## 四、推荐路线
# 报错急救路线

适合：已经失败、不知道错在哪里
难度：低
耗时：10-30 分钟

## 准备
- 完整报错文字
- 系统信息
- 执行过的命令

## 步骤
1. 停止重复安装
2. 复制完整报错
3. 到报错库搜索关键词
4. 生成国产 AI 排错提示词
5. 只执行一个修复方案
6. 记录是否成功

## 风险提醒
反复叠加修复命令、漏掉原始错误、把 API Key 粘贴给别人。

# Windows 小白标准路线

适合：Windows + 完全小白 + 不懂终端
难度：低到中
耗时：30-60 分钟

## 准备
- Windows Terminal 或 PowerShell
- 网络稳定
- ChatGPT 账号或 API Key
- 英文路径项目文件夹

## 步骤
1. 确认系统版本和管理员权限
2. 安装/打开 Windows Terminal
3. 执行官方 PowerShell 安装脚本
4. 运行 codex 并完成登录
5. 创建英文路径项目文件夹
6. 运行测试任务并记录结果
7. 导出配置报告

## 风险提醒
PowerShell 权限、PATH 未刷新、公司电脑限制、中文路径。
