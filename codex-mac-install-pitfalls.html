安装前安全提醒

  • 不要把 API Key、Token、邮箱验证码、SSH 私钥发给陌生人。
  • 不要执行来源不明的删除、重装、权限提升命令。
  • 安装前把重要项目、桌面文件、客户资料做好备份。
  • 报错截图前先遮挡邮箱、密钥、Token、个人路径。
  • 不要购买来路不明账号，不做规避风控或绕过平台限制的操作。
