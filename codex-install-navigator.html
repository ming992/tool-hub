配置状态

安全清单

  • 不要把 API Key 发给陌生人。
  • 不要把 Key 写进 HTML/JS 前端代码。
  • 不要上传到 GitHub、网盘公开链接或截图。
  • 发现泄露后立即删除/轮换密钥。
  • 用环境变量或系统登录方式配置，优先不明文保存。
setx OPENAI_API_KEY "你的真实Key不要粘贴到本工具里"

设置后建议重新打开终端，再运行测试命令。

export OPENAI_API_KEY="你的真实Key不要粘贴到本工具里"

要长期生效，可以写入 shell 配置文件，但请谨慎保管。
